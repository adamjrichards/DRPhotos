D113-000 05-05-13 - Pit stop, Orangeville.
Stopped and bought camera memory cards. There just happened to be a store near where we parked for lunch that sold discounted, because obsolete, camera memory cards. I seem to have spent a chunk of my life backing technologies that didn't pan out. First it was the 120 size film format that I used in Europe. 35mm, my eventual choice, would have been more practical from the start. Next, I backed versions of the Microsoft DOS operating system while everyone else was switching to Windows. More lately, the XD format memory cards of my first digital camera have given way to the nearly universal SD format.

Birding walk. Derek goes abstract.
Self-defense when you're surrounded by birders with binoculars, telescopes and long lenses and you have only an auto-everything pocket camera. You have to do something investigative.
